== Description ==

Contact Form Extender for Divi Builder extends the default Divi contact form module with useful extra features.

This plugin helps you:

* Add file upload field to Divi contact form
* Save contact form submissions in database
* Manage form entries easily from WordPress

More advanced features will be added in future updates.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` or install via WordPress Plugins screen.
2. Activate the plugin through the Plugins menu.
3. Use the Divi Contact Form module as usual.
4. Extra features will be available automatically.

== Frequently Asked Questions ==

= Does this plugin work with Divi Builder? =
Yes, this plugin works with the Divi Contact Form module.

= Does it save form submissions in database? =
Yes, submitted form entries are saved in the WordPress database.

= Is file upload supported? =
Yes, you can add a file upload field to the Divi contact form.
